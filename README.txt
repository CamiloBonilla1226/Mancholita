Mancholita
--> Este proyecto es una solución integral para la gestión de una tienda de ropa online. 
El sistema permite a los clientes navegar por un catálogo dinámico y realizar pedidos,
mientras ofrece a los administradores una plataforma robusta para controlar inventarios y ventas.

🚀 Funcionalidades Principales
Catálogo: Navegación por categorías jerárquicas (Hombre/Mujer -> Subcategorías) y búsqueda de productos.
Gestión de Pedidos: Registro ágil de compras con generación de facturas internas.
Panel Administrativo: Control total de productos (CRUD) y estados de stock.
Reportes en Excel: Exportación de órdenes diarias y generales para contabilidad.
Seguridad: Endpoints administrativos protegidos mediante autenticación robusta.

🛠️ Stack Tecnológico
Backend: Java 17, Spring Boot 3, Spring Security, JPA/Hibernate.
Base de Datos: MySQL 8.0.
Reportes: Apache POI.
Arquitectura: Diseño por capas (Controller-Service-Repository).
